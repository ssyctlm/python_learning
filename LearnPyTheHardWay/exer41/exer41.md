# ex41 Leraning To Speak Object Oriented

## Word Drills

- class: Tell Python to make new kind of things.
- object: Two meanings: the most basic kind of thing, and any instance of some thing.
- instance: What you get when you tell Pythong to create a class.
- def: How you define a function inside a class.
- self: Inside the function in a class, self is a variable for the instance/object being accessed
- inheritance: The concept that one class can inherit traits form another class, much like you and your parents.
- composition: The concept that a class can be composed of other classes as parts, much like how a car has wheels.
- attributes: A property classes have that are from composition and are usually variables.
- is-a: A phrase to say that something inherits from another, as in a Salmon is-a Fish.
- has-a: A phrase to say that something is composed of other things or has a trait, as in a Salmon has-a mouth

## Phrase Drills

** class X(Y)**
  "Make a class named X and that is-a Y"
  "创建一个叫做X的类，并且它是一个Y"

** class X(object): def __init__(self,j) **
  "class X has-a __init__ that takes self and j parameters"
  "类X有初始化，并会接受self和J为变量"
    
** class X(object): def M(self,j)**
  "class X has-a function named M that takes self and j parameters"
  "类X有一个名叫M的方法，并且会接受self和J变量"

** foo = X()**
  "Set foo to an instatnce of class X"
  "设 foo 为一个X类的实例"

** foo.M(J) **
  "From foo get the M function, and call it with parameters self,J"
  "从 foo 实例中获得M方法，并且使用slef和J变量来调用它"
    
** foo.K = Q **
  "From foo get the K attribute and set it to Q."
  "从 foo 实例中获取属性K 并将它设为Q"
